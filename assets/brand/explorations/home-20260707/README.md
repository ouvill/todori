# Todori Home Today Explorations - 2026-07-07

## A. TickTick direction

Efficiency-first layout with compact rows, clear Overdue / Today grouping, and a persistent quick-add bar.
The slightly opened swipe row explores how dense task management can reveal actions without adding permanent chrome.

## B. Todoist direction

Ultra-minimal layout built around whitespace, typography, check circles, and task titles.
Metadata is intentionally quiet so the screen feels fast, lightweight, and almost text-only.

## C. Current structure polish

Keeps the current Todori structure: large serif Today header, date, single Tasks panel, and bottom Add task pill.
The exploration tightens header spacing, softens metadata pills, and raises surface quality while preserving the existing mental model.
